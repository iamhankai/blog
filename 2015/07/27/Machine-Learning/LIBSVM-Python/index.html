简介

LIBSVM是台湾大学林智仁教授等人开发的SVM软件包，目前有C、Java、Python、Matlab、C#、Ruby、R、Perl、Common LISP、Labview等数十种语言版本，供不同需求的开发人员使用。
以前在Python中都是使用scikit-learn中的SVM，实习中上司让我用LIBSVM，所以就学习了下。今天来介绍一下如何在Python中使用LIBSVM，其他版本的使用都大同小异。其实这些在下载下来的软件包的README文件中都有介绍，有不明白的地方可以翻翻README。

配置LIBSVM

首先，到官网或LIBSVM的github主页下载LIBSVM压缩包，解压到某目录，如E:/libsvm-3.20/。
然后，看一下自己Python的版本，32位的容易配置，不用编译（因为林已经帮你编译好了），直接在Python代码中引用就可以了；64位的需要自己再编译一下LIBSVM，生成64位的动态链接库，参考SVM学习笔记（2）LIBSVM在python下的使用。

使用LIBSVM

LIBSVM提供给Python两种SVM模块，一是svmutil.py，偏高层，更易于使用，适合入门者使用，二是svm.py，偏底层，需要读者访问底层函数，更灵活，对于入门者来说稍显复杂。我们这里使用svmutil模块。

其他

svmutil模块包含的函数

svmutil模块主要包含了如下函数

svm_train() : train an SVM model
svm_predict() : predict testing data
svm_read_problem() : read the data from a LIBSVM-format file.
svm_load_model() : load a LIBSVM model.
svm_save_model() : save model to a file.
evaluations() : evaluate prediction results.
