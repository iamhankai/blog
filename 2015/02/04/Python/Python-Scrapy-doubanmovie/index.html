Scrapy是Python开发的一个快速,高层次的屏幕抓取和Web抓取框架，用于抓取Web站点并从页面中提取结构化的数据。在批量抓取大量的内容时，使用起来显得尤为方便快捷。
以下操作均在Windows 7 专业版下进行，其他系统仅供参考。

Scrapy安装

Scrapy安装很繁琐，安装教程网上有很多，但未必都有效，我试了很多，最后按照下面这个才安装成功：
网络爬虫（蜘蛛）Scrapy,Python安装！

MySQL和MySQL-python安装

本次试用，我们用Scrapy获取豆瓣电影Top 250的电影信息，存储在MySQL中，所以要安装两个东西：

  • MySQL（注意，截至本文撰写，Python只支持MySQL5.5及其以下版本，我们安装MySQL5.5）
  • MySQL-python
